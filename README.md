# Price Watcher

A personal site to keep track of price changes for products you want to monitor.

The app lives in the **`price-watcher`** folder. See [`price-watcher/README.md`](price-watcher/README.md) for setup and run instructions.

```bash
cd price-watcher
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
